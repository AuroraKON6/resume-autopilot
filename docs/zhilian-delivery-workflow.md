# 智联招聘 OCR 投递工作流

依据 `docx/智联招聘的登录界面.docx` 设计。四个平台的审查层保持一致：复用同一个简历画像，用 DeepSeek 审查当前岗位详情；不同平台只区分搜索、进入详情、投递和返回方式。

## 页面状态

1. 未登录或安全阻塞
   - 信号：右上角 `登录/注册`，或右侧出现 `验证码登录/注册`、手机号、短信验证码、获取验证码、用户协议/隐私政策
   - 动作：`need_user`，不自动登录、不填短信、不处理验证码。

2. 搜索页
   - 信号：`搜索职位、公司`、`职位类别`、`公司行业`、城市/筛选项、蓝色搜索按钮
   - 动作：输入目标关键词并点击搜索。

3. 搜索结果页
   - 信号：`智能匹配`、`薪酬最高`、`最新发布`、岗位卡、`立即投递`、`立即沟通`
   - 动作：不要点列表卡片里的 `立即投递`；点岗位标题文字进入详情页。

4. 岗位详情页
   - 信号：`职位描述`、`工作职责`、`任职要求`、`工作地点`、`职位发布者`、`公司基本信息`、`立即投递`
   - 动作：抽取详情页主体文本给 DeepSeek 审查。

5. 投递动作
   - 信号：详情页 `立即投递`
   - 动作：这是最终投递动作，必须通过 DeepSeek 门禁。

6. 成功或已投递
   - 信号：`投递成功`、详情页按钮变为 `已投递`、`简历投递成功`、`您已投递该职位`
   - 动作：记为成功，然后关闭当前浏览器标签页，使上一标签的搜索结果页重新激活，再继续下一岗位。
   - 关闭方式：优先点击当前 Chrome 标签页顶部 `X`，或使用 `Ctrl+W`。不要点击左边/上一页的页面内容来返回。

## 禁止动作

- 不自动处理登录、扫码、短信、验证码、人机验证、滑块。
- 不在搜索结果页直接点击 `立即投递`。
- 不默认点击 `立即沟通`、`收藏`、`微信扫码分享`、`APP`、`举报`。
- 不上传、选择、修改简历；简历只用于生成一次可复用画像。
- 未通过 DeepSeek 审查时不点击详情页 `立即投递`。

## 快速模式预留

后续快速模式可以跳过 DeepSeek，仅按岗位标题/关键词做粗匹配；当前基础流程不启用快速模式，仍保持四站统一 DeepSeek 审查。
